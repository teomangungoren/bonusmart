# BonusMart Frontend

Modern React application for BonusMart e-commerce platform.

## Tech Stack

- React 18
- TypeScript
- Material-UI (MUI)
- Redux Toolkit
- React Router
- Axios

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000)

### Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Project Structure

```
src/
├── components/     # Reusable components
├── pages/          # Page components
├── services/       # API services
├── store/          # Redux store
├── hooks/          # Custom hooks
├── types/          # TypeScript types
└── constants/      # Constants
```

## API Integration

The app connects to the API Gateway at `http://localhost:8081`.

## Features

- Product listing and search
- Product details
- Category navigation
- User authentication
- Responsive design
